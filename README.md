## How to install

Install all dependencies

```
npm install
```

Create the database using this [SQL](./database.sql) code

Change the connection string  of your database at your .env file

## Env file

```
// Example
baseURL = "http://localhost:3005"
connectionString = ""
```

## Scripts

```
npm run dev
```

```
npm run test
```
